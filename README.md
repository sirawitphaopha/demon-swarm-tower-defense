# ⚔️ Demon Swarm TD

เกมแนว **Tower Defense (วางป้อมป้องกัน)** เล่นบนเบราว์เซอร์ ธีมทุ่งหญ้าสดใส
วางป้อมขวางทางแมลงปีศาจ อย่าให้หลุดออกทางขวาเกิน 20 ตัว

เขียนด้วย HTML + Canvas + JavaScript ล้วน **ไฟล์เดียวจบ** (`index.html`) ไม่ต้องติดตั้งอะไร

---

## 🎮 วิธีเล่น

1. กด **เริ่มเกม** → จะมีช่วง **เตรียมตัว** ให้วางป้อมก่อนศัตรูบุก
2. เลือกป้อมจากแถบซ้าย แล้วคลิกบนสนามเพื่อวาง
3. วางป้อมต่อกันเป็น **เขาวงกต** เพื่อบังคับให้ศัตรูเดินอ้อม (ยิ่งทางยาว ยิ่งยิงได้นาน)
4. ฆ่าศัตรูได้เงิน เอาไปวางป้อมเพิ่มหรืออัปเกรด
5. คลิกป้อมที่วางแล้วเพื่อ **อัปเกรด / ขาย / เปลี่ยนเป้าเล็ง**
6. อย่าให้ศัตรูหลุดออกขวาเกิน 20 ตัว มิฉะนั้นแพ้

### ⌨️ คีย์ลัด
- `1`–`6` เลือกป้อม
- `Space` หยุด/เล่นต่อ
- `Esc` ยกเลิกการเลือก
- คลิกขวา ยกเลิกการเลือก

---

## ✨ ฟีเจอร์

- **6 ป้อม** อัปเกรดต่อเชน: หินขว้าง → หอธนู → ครอสโบว์ → ปืนใหญ่(AOE) → ปืนกล → เลเซอร์
- **6 ศัตรู** หนอน / ด้วง / มอดเกราะ / แตนปีศาจ(บินข้ามป้อม) / แมงมุมผี / ราชาแมลง(บอส)
- **โหมดเล็งเป้า** 4 แบบ: หน้าสุด / หลังสุด / เลือดมาก / เลือดน้อย
- **2 โหมดเกม**: 10 เวฟ (เคลียร์ครบคือชนะ) และ **ไม่สิ้นสุด** (บุกไม่หยุด ยิ่งสูงยิ่งโหด)
- **3 ระดับความยาก** ง่าย / ปกติ / ยาก
- **ระบบหาทางจริง (flow-field BFS)** ศัตรูเดินอ้อมป้อม/สิ่งกีดขวาง ระบบกันวางป้อมปิดทางตาย
- **4 ธีมแมพ** ทุ่งหญ้า / ป่าสน / ทุ่งดอกไม้ / โอเอซิส — พื้นหญ้าเขียวสดทุกธีม
- **สิ่งกีดขวางสุ่ม** ต้นไม้ / แอ่งน้ำ(ทรงธรรมชาติ) / หิน — สุ่มตำแหน่งใหม่ทุกเกม
- **🛠️ เครื่องมือสร้างแมพ** ลากเมาส์วาดแมพเอง บันทึกได้หลายแมพ เอามาเล่นได้
- **เสียงในตัว** สร้างจาก Web Audio (ไม่ต้องโหลดไฟล์เสียง) เปิด/ปิดได้
- **บันทึกสถิติสูงสุด** ในเครื่อง (localStorage)
- ปรับความเร็วเกม x1 / x2 / x3

## ▶️ วิธีรัน

เปิดเว็บ static ธรรมดา เพราะเป็นไฟล์เดียว:

```bash
# วิธีที่ 1: เปิดไฟล์ตรงๆ
เปิด index.html ด้วยเบราว์เซอร์

# วิธีที่ 2: รันผ่านเซิร์ฟเวอร์ในเครื่อง (แนะนำ)
npx serve .
# หรือ
python -m http.server 8000
```

> หมายเหตุ: เสียงและการบันทึกแมพ/สถิติ ทำงานได้ดีที่สุดเมื่อเปิดผ่าน `http://` (เซิร์ฟเวอร์) ไม่ใช่ `file://`

## ☁️ Deploy (Cloudflare Pages)

โปรเจกต์มี `wrangler.jsonc` ตั้งค่าไว้แล้ว เป็นเว็บ static ล้วน
- Build command: เว้นว่าง
- Output directory: `/` (root)
- ไม่ต้องตั้ง environment variables ใดๆ
